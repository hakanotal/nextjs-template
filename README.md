## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn dev

# build
$ yarn build

# production mode
$ yarn start
```

## Tech Stack
- [TypeScript](https://www.typescriptlang.org/)
- [React](https://reactjs.org/)
- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Mantine UI](https://mantine.dev/)
- [Zod](https://zod.dev/)
